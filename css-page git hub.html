Conceptos clave
🎨

Selectores

Apuntan a elementos HTML por etiqueta, clase (.clase), ID (#id) o relación entre elementos.

📦

Box Model

Todo elemento es una caja: content → padding → border → margin. Entenderlo es esencial para el layout.

🔲

Flexbox

Sistema de layout unidimensional. Alinea y distribuye elementos en filas o columnas con facilidad.

🔳

Grid

Sistema de layout bidimensional. Crea layouts complejos con filas y columnas simultáneamente.

✨

Animaciones

Con @keyframes y transition das vida a tus interfaces sin necesidad de JavaScript.

📱

Responsive

Media queries (@media) adaptan el diseño a cualquier tamaño de pantalla: móvil, tablet, desktop.

Ejemplo de código
/* Variables CSS y Flexbox */
:root {
  --color-primario: #264de4;
  --espaciado: 1rem;
}

.tarjeta {
  display: flex;
  flex-direction: column;
  gap: var(--espaciado);
  padding: 24px;
  border-radius: 12px;
  background: linear-gradient(135deg, #1a1a2e, #16213e);
  transition: transform 0.3s ease;
}

.tarjeta:hover {
  transform: translateY(-8px);
}

💡 ¿Sabías que?

CSS fue propuesto por Håkon Wium Lie en 1994 porque en ese momento los diseñadores no tenían control visual sobre sus páginas. Hoy, con CSS Grid y Custom Properties, podemos crear interfaces que antes solo eran posibles con aplicaciones nativas de escritorio.
