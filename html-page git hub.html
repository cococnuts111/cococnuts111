← Volver al perfil
Conceptos clave
🏗️

Estructura

HTML define la estructura y el significado del contenido mediante etiquetas anidadas en un árbol llamado DOM.

🏷️

Etiquetas semánticas

<header>, <main>, <section>, <article>, <footer> dan significado al contenido más allá del diseño.

📝

Formularios

Con <form>, <input>, <select> y <button> capturamos datos del usuario de forma nativa.

🔗

Hipervínculos

La etiqueta <a href=""> es la esencia de la web: conectar páginas y recursos entre sí.

🖼️

Multimedia

HTML5 trajo <video>, <audio> e <canvas>, permitiendo multimedia sin plugins externos.

♿

Accesibilidad

Atributos como alt, aria-label y roles hacen que el contenido sea accesible para todos.

Ejemplo de código
<!-- Estructura básica de una página HTML5 -->
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi primera página</title>
</head>
<body>
  <header>
    <h1>¡Hola, mundo!</h1>
  </header>
  <main>
    <p>Esta es mi primera página web.</p>
    <a href="https://developer.mozilla.org">
      Aprender más →
    </a>
  </main>
</body>
</html>

💡 ¿Sabías que?

HTML fue creado por Tim Berners-Lee en 1991. La primera versión solo tenía 18 etiquetas. Hoy HTML5 tiene más de 100, y cada una tiene un propósito semántico específico que ayuda tanto a los navegadores como a los motores de búsqueda a entender tu contenido.
