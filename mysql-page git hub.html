MySQL Database

El sistema de bases de datos relacionales más popular del mundo. Organiza, consulta y protege tu información.

← Volver al perfil
Conceptos clave
🗃️

Tablas

Los datos se organizan en tablas con filas y columnas, similar a una hoja de cálculo pero con estructura estricta.

🔑

Primary Key

Identificador único para cada fila. Generalmente un campo AUTO_INCREMENT llamado id.

🔗

Foreign Key

Relaciona tablas entre sí. Un pedido tiene un cliente_id que apunta a la tabla de clientes.

🔍

SELECT y JOINs

Consultas para obtener datos. Los JOINs combinan datos de varias tablas en una sola consulta.

📝

CRUD

Las 4 operaciones básicas: CREATE (INSERT), READ (SELECT), UPDATE, DELETE.

🛡️

Transacciones

COMMIT y ROLLBACK garantizan que operaciones críticas ocurran completas o no ocurran en absoluto.

Ejemplo de código
-- Crear tabla de estudiantes
CREATE TABLE estudiantes (
  id        INT AUTO_INCREMENT PRIMARY KEY,
  nombre    VARCHAR(100) NOT NULL,
  email     VARCHAR(150) UNIQUE,
  creado_en TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Insertar un registro
INSERT INTO estudiantes (nombre, email)
VALUES ('Josué Enrique', 'josue@email.com');

-- Consultar con JOIN
SELECT e.nombre, c.titulo
FROM estudiantes e
JOIN cursos c ON e.id = c.estudiante_id
WHERE c.aprobado = 1;

💡 ¿Sabías que?

MySQL fue creado en 1995 por Michael Widenius y David Axmark. El nombre "My" viene del nombre de la hija de Widenius. Hoy es usado por empresas como Facebook, Twitter, YouTube y Netflix para manejar billones de registros diariamente. Es el motor de bases de datos más popular según múltiples rankings.
