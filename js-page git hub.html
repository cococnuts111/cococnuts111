Conceptos clave
🔄

Variables y tipos

let, const y var almacenan datos. JS maneja strings, numbers, booleans, arrays y objetos dinámicamente.

⚙️

Funciones

Bloques de código reutilizables. Arrow functions (=>) son la forma moderna y concisa de definirlas.

🌐

DOM

Document Object Model: JS puede leer y modificar cualquier elemento HTML en tiempo real.

⏳

Async / Await

Permite esperar respuestas de servidores sin congelar la página. Esencial para APIs y fetch().

📦

Arrays y Objetos

map(), filter(), reduce() transforman datos. Los objetos agrupan propiedades y métodos relacionados.

🎯

Eventos

addEventListener() escucha clicks, teclas, scroll y cualquier acción del usuario para responder a ella.

Ejemplo de código
// Función asíncrona que obtiene datos de una API
const obtenerUsuario = async (id) => {
  try {
    const respuesta = await fetch(`/api/usuarios/${id}`);
    const datos = await respuesta.json();

    // Desestructuración de objeto
    const { nombre, email, edad } = datos;

    return { nombre, email, edad };
  } catch (error) {
    console.error("Error:", error);
  }
};

💡 ¿Sabías que?

JavaScript fue creado por Brendan Eich en solo 10 días en 1995 para Netscape. A pesar de su origen apresurado, se convirtió en el lenguaje más usado del mundo. Hoy corre en navegadores, servidores (Node.js), apps móviles (React Native) e incluso microcontroladores.
